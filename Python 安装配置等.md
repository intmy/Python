## 安装python3

    sudo apt-get install python3
    
## 运行

    python3
    
## 检查版本

    python3 --version
    
## 创建静态资源服务器（http的根目录即为当前命令的执行目录）

    python3 -m http.server

## 安装pip

    sudo apt-get install python-pip python-dev build-essential
    sudo pip install --upgrade pip

## 安装pip3

    sudo apt-get install python3-pip
    
## 运行pip3

    pip3
