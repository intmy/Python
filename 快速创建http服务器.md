## Python 3.x

    python -m http.server <端口号,如不提供默认是8000>
    
    (当操作系统存在多个python版本，可能需要制定python版本，这时可以把以上代码改为类似 python3 -m http.server 这种)
    
  
## Python 2.x

    python -m SimpleHTTPServer 8080
  
    (自定义端口号最好大于1024)
